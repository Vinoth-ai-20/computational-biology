# Module 19 — Research Seminar and Paper Reading

**Tier:** Ongoing/Cross-cutting — this is a weekly habit, not a finite module
**Week budget:** ~2 hours/week for the full 12 months
**Track A relevance:** The most common bioinformatics interview pattern: handed a paper cold and asked for a "back-of-the-envelope" explanation. Pass-3 practice is direct interview rehearsal.
**Track B relevance:** PhD applications require a coherent research-interest narrative and demonstrated independent reading. The paper-notes/ log is the evidence.

## What this module is

A practice framework, not a content module. Each notebook covers one skill: how to skim, how to identify methods, how to evaluate claims, how to present in a journal club, how to write a literature review. The notebooks are tools for practice, not lectures to be memorised.

This module runs as a continuous weekly practice throughout the entire 12-month programme. It is not "completed" — it accumulates. The output is not a final notebook but a growing `paper-notes/` directory at the repository root, a rising Pass-3 success rate, and the ability to walk into any bioinformatics interview and explain an abstract you have never seen before.

## Weekly cadence

Each week: read one paper (Pass 1 + Pass 2), attempt Pass 3, log in `paper-notes/`. Monthly: pick one paper from a prior module's `papers.md` for a full journal-club-style presentation using NB06.

## Track A connection (explicit)

The single most common bioinformatics RA/JRF interview pattern identified in current hiring research: an interviewer hands you an abstract and asks "can you walk me through this?" Pass-3 practice (close the paper, re-explain it unaided) is direct rehearsal for exactly this. Treat NB10 as interview simulation, not academic exercise.

## Track B connection (explicit)

European PhD applications — particularly EMBL EIPP, IMPRS, CSH Vienna, Wageningen — evaluate candidates on demonstrated independent engagement with the literature. The `paper-notes/` log (especially the Pass-3 binary: "reconstructed unaided Y/N") is concrete, public, dateable evidence of independent scientific reading. Build it consistently from Week 1.
