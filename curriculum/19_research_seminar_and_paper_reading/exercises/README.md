# Module 19 — Exercises

Exercises for this module are practice tasks, not problem sets. They are posed in each notebook and answered through actual paper reading and logging, not by writing code in an exercise file.

## Exercise structure

Each notebook (NB01–NB12) poses exercises at the end. These are:
- **Reading tasks:** Read a specified paper using the Three-Pass Method and log the result in `paper-notes/`.
- **Application tasks:** Apply the notebook's tool (MethodsChecker, PresentationAudit, etc.) to a real paper you have chosen.
- **Reflection tasks:** Answer in plain English, no jargon — the Feynman Technique applied to paper reading.

## Exercise log

Track completed exercises in your weekly `progress/` log. Format:

```
Week 03:
- NB01 Exercise 1: Three-Pass on Needleman-Wunsch 1970. Pass-3 success: Y.
- NB02 Exercise 1: MethodsChecker on STAR methods section. Flags raised: 2 (no random seed, no accession).
```
