# Module 11 — Mini Projects

**Tier 3 — No mini project required.**

Mini projects are mandatory only for Tier 1 (Master) and Tier 2 (Working competence)
modules. Module 11 is Tier 3 — Survey.

If you complete the module ahead of schedule or a real signal (PhD programme, job
posting) indicates structural biology depth is needed, consider:

- **Optional MP01:** Build a complete structure analysis pipeline on a real PDB entry:
  download → parse → Ramachandran plot → contact map → secondary structure assignment
  → radius of gyration → compare with DSSP output using Biopython
- **Optional MP02:** Fetch two AlphaFold2 predictions for orthologous proteins and
  compute their Cα RMSD; compare pLDDT scores between the two

Neither of these should be started unless a genuine signal justifies the time cost.
The Tier 3 depth commitment is the baseline; expanding it requires a quarterly
checkpoint rationale (see CLAUDE.md §11).
