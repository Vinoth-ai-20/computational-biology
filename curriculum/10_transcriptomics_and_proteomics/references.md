| Tool | URL | Notes |
|------|-----|-------|
| scanpy | scanpy.readthedocs.io | Primary scRNA-seq framework; tutorials are excellent |
| anndata | anndata.readthedocs.io | Data structure backing scanpy |
| PBMC 3k tutorial | scanpy.readthedocs.io/en/stable/tutorials/basics/clustering.html | The canonical scanpy walkthrough |
| Seurat (R, for reference) | satijalab.org/seurat | Most widely used in practice; scanpy is the Python equivalent |
| umap-learn | umap-learn.readthedocs.io | UMAP reference implementation |
| Perseus (proteomics) | maxquant.net/perseus | Standard proteomics analysis platform |
| MaxQuant | maxquant.org | Label-free and TMT proteomics quantification |
