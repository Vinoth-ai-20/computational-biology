# Datasets — Module 02: Mathematics for Biology

Module 02 uses only synthetic or simulated data — no external datasets are required.

## Synthetic data generated in notebooks

| Dataset | Generated in | Description |
|---------|-------------|-------------|
| Bacterial growth curve | NB04 | Simulated logistic growth with noise |
| Logistic map time series | NB05 | Discrete logistic map for r ∈ [2, 4] |
| Predator-prey dynamics | NB09 | Lotka-Volterra numerical solution |
| Food web graph | NB10 | 6-node synthetic food web |

All datasets are generated with fixed seeds for reproducibility.
