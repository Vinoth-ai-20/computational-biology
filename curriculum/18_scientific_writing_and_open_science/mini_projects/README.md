# Module 18 — Mini Projects

## MP01 — Write a complete methods section + abstract for a prior result

**Context:** This mini-project integrates writing skills across the full module.
You select any substantive result from Modules 09–15 that produced a real
computational output (fitted model, trained classifier, simulation result).
You then write the two sections that reviewers read most critically: the abstract
and the methods.

**Tasks:**
1. **Choose a result** from a prior module. Examples:
   - Module 13 NB10: Random forest for gene expression classification
   - Module 15 MP01: Epidemic model comparison
   - Module 15 NB09: SIR parameter estimation by MCMC
   - Module 14 MP01: DeepBind CNN vs. k-mer+SVM
2. **Write a 200-word abstract** using the 6-sentence template (NB02).
   Get the motivation, gap, approach, key result, and significance into exactly
   6 sentences — no more, no fewer.
3. **Write a 400-word methods section** sufficient for full replication (NB04 checklist):
   - Dataset/synthetic data generation
   - All algorithm steps with parameters
   - Software versions and random seeds
   - Validation strategy
   - Statistical analysis
4. **Produce one publication-quality figure** (NB07 standards) that accompanies
   the methods section.
5. **Write a 50-word data and code availability statement** referencing this repository.

**Portfolio value:** A polished abstract + methods section is the most reusable
writing artefact for PhD applications — supervisors and programme committees often
ask for a writing sample, and this is exactly the right format.

**Deliverable:** `mini_projects/MP01_methods_and_abstract.md` (or `.tex`) + figure file.
