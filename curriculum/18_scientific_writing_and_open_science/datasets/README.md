# Module 18 — Datasets

Most notebooks in this module use text data (papers, abstracts) and figures
generated from prior modules. A few notebooks use synthetic text corpora for
quantitative analysis.

| ID | Dataset | Source | Used in | Notes |
|----|---------|--------|---------|-------|
| DS01 | Sample abstracts corpus (synthetic) | Generated in NB02 | NB02, NB12 | 50 synthetic computational biology abstracts for analysis exercises |
| DS02 | Ten Simple Rules papers corpus | PLOS Computational Biology | NB01, NB09 | Plain-text extracts; verify open-access license (CC BY) before use |
| DS03 | LaTeX document templates | In-notebook | NB11 | Generated; MIT licensed |
| DS04 | Publication figure templates | Generated in NB07 | NB07, NB12 | PNG/PDF outputs from matplotlib; no external data needed |
| DS05 | EMBL/IMPRS application examples (anonymized) | Publicly available | NB10 | Collect from programme websites; check terms before committing |

## Notes on text data

Never commit copyrighted paper PDFs to this repository (see project constitution
§9, rule 4). Save only:
- Citation metadata (author, title, DOI, year)
- Your own notes and annotations (in `paper-notes/`)
- Short quotations for educational illustration (fair use / fair dealing)

## Generated outputs that can be committed

- `.png` / `.pdf` figures generated by NB07
- `.tex` + `.pdf` from NB11 examples
- Draft text files from exercises (your own original writing)
