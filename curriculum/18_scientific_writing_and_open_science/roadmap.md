## Part 1 — Paper Structure and Argumentation (Week 1)

| # | Notebook | Key concepts |
|---|----------|-------------|
| 01 | `01_anatomy_of_a_scientific_paper.ipynb` | IMRaD structure; scientific argument flow; abstract → introduction → methods → results → discussion; reading strategy for comprehension vs. citation mining |

## Completion criteria (Tier 2)

- Can write a 200-word structured abstract that passes the 6-sentence template
- Can produce a publication-quality figure with proper labels, legend, and colour scheme
- Can identify the five FAIR principles and describe what each requires in practice
- Can write a 500-word PhD research proposal that states a question, approach, and expected outcomes
- Can compile a LaTeX document with equations, figures, tables, and a reference list
- Can write a constructive peer review in the standard format
