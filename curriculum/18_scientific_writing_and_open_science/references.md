# Module 18 — References

## Core textbooks and guides

| Title | Authors | Why |
|-------|---------|-----|
| *The Craft of Scientific Writing* (4th ed.) | Michael Alley | The standard reference for scientific prose; focus on clarity and precision over formality |
| *Writing Science* | Joshua Schimel | Narrative structure in scientific writing; why storytelling techniques produce better papers, not worse ones |
| *A PhD Is Not Enough!* | Peter Feibelman | Career communication beyond papers: talks, posters, job talks, networking — still highly relevant for early-career researchers |
| *The Art of Writing Proposals* | Adam Przeworski & Frank Salomon | Short (~30 pages); proposal-writing craft that applies directly to PhD applications and grants |
| *Ten Simple Rules for…* series (PLOS Computational Biology) | Various | Multiple relevant papers: writing papers, PhD theses, figures, computational analyses, reproducible research |

## Key papers to read as writing models

| Paper | Why |
|-------|-----|
| Lander et al. (2001) "Initial sequencing and analysis of the human genome" *Nature* | Example of how to write a results section for a massive computational analysis; clear section headers, figure-first |
| Almeida et al. (2021) "A unified catalog of 204,938 reference genomes from the human gut microbiome" *Nature Biotechnology* | Modern bioinformatics paper structure; methods supplementary; data/code availability statement |
| Yosef et al. (2013) "Dynamic regulatory network controlling TH17 cell differentiation" *Nature* | Strong introduction funnel; crisp gap statement; model paper for systems biology writing |

## Tools

| Tool | Role |
|------|------|
| [Overleaf](https://www.overleaf.com/) | Collaborative LaTeX editor; free tier sufficient; cloud compilation |
| [Zotero](https://www.zotero.org/) | Reference manager; Better BibTeX plugin for BibTeX export |
| [bioRxiv](https://www.biorxiv.org/) | Preprint server for biology; free; DOI assigned immediately |
| [Zenodo](https://zenodo.org/) | Data/code archival; DOI for any research output; integrates with GitHub |
| [OSF (Open Science Framework)](https://osf.io/) | Pre-registration, data sharing, lab notebook — all in one |
| [ORCID](https://orcid.org/) | Persistent author identity; required by most European funders |
| [Scimago Journal Rankings](https://www.scimagojr.com/) | Check journal quartile and impact factor before submitting |
| [Grammarly](https://www.grammarly.com/) / [LanguageTool](https://languagetool.org/) | Grammar and style checking (use as a tool, not a substitute for craft) |
| [pandoc](https://pandoc.org/) | Convert between Markdown, LaTeX, Word, HTML |
| [matplotlib](https://matplotlib.org/) + [seaborn](https://seaborn.pydata.org/) | Publication-quality Python figures |
