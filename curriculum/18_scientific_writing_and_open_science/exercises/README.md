# Module 18 — Exercises

These exercises require actual writing, not just code. Submit each in
`exercises/answers/EXX_your_answer.md`. Aim for the specified word count —
being 20% over is as bad as being 20% under.

| Exercise | Paired notebook | Task | Word target |
|----------|----------------|------|------------|
| E01 | NB01 — Paper anatomy | Choose any bioinformatics paper from the last 3 years. Map its structure to IMRaD. Write one sentence per section describing the key claim made. Does the paper's story hold together? | 200 words |
| E02 | NB02 — Abstracts | Take the abstract of the paper from E01. Strip it of all content. Now reconstruct it using the 6-sentence template from scratch (before re-reading the original). Compare your version to the original. | 200 words |
| E03 | NB03 — Introductions | Write an introduction paragraph (the first one only — the broad context) for a paper based on your Module 13 random forest gene expression work. Funnel: field → problem → your approach. Do not mention your methods yet. | 150 words |
| E04 | NB04 — Methods | Write a complete methods section for the Gillespie SSA implementation from Module 15 NB02. Another computational biologist should be able to reproduce your simulation exactly. Include: programming language + version, algorithm steps, parameters used, random seed, and how results were validated. | 300 words |
| E05 | NB05 — Results | Write the results paragraph for the Module 15 MP01 epidemic model comparison (ODE vs. network vs. ABM SIR). Report the peak infected fraction and final epidemic size for each model, with appropriate quantification. Do not interpret — only report. | 200 words |
| E06 | NB06 — Discussion | Write a limitations paragraph for the Module 15 tumor growth model (NB11). What assumptions does it make that are biologically unrealistic? What would a better model require? How would you test whether the limitation affects the conclusions? | 200 words |
| E07 | NB07 — Figures | Take the 6-panel figure from Module 15 NB01 (ODE biological models) and recreate it as a publication-quality single figure. | E09 | NB09 — Peer review | Read any open-review paper from eLife (look for "decision letter" on any eLife paper page). Write a 200-word summary: what were the reviewers' main concerns? How did the authors respond? Was the response adequate? | 200 words |
| E10 | NB10 — PhD statement | Write a 500-word personal statement for a PhD application to EMBL or an IMPRS programme.
