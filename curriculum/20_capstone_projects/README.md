# Module 20 — Capstone Projects

**Tier:** 1 — Master
**Week budget:** 4 weeks (Months 11–12)
**Track A relevance:** The capstone projects ARE the portfolio — polished, documented, reproducible analyses that demonstrate everything a Track A employer needs to see: real data, working code, publication-quality figures, written results.
**Track B relevance:** The capstone is the centrepiece of the PhD application: a public, working repository with ≥1 strong written artifact, a coherent research-interest narrative, and evidence of independent scientific work.

## What this module produces

Three polished capstone projects, each targeting a different sub-field, plus a complete portfolio review and PhD-application package.

**CP01 — Genomics pipeline:** End-to-end RNA-seq differential expression analysis using synthetic negative-binomial count data (DESeq2-equivalent Python implementation, volcano plot, GO enrichment).

**CP02 — Machine learning for biology:** TF binding site prediction — full comparison of k-mer+SVM baseline vs CNN, proper 5-fold cross-validation, publication-quality figures, and a complete written abstract + methods section.

**CP03 — Simulation study:** Epidemic model comparison (ODE SIR vs network SIR vs spatial ABM), MCMC parameter estimation, AIC model comparison.

## How this connects to the 12-month programme

Every prior module feeds into a capstone. The capstones integrate the 13-step teaching order (§4.1), the statistical reporting standard (Module 18 NB05), the publication-quality figure standard (Module 18 NB07), and the abstract + methods writing skills (Module 18 NB02–04).
