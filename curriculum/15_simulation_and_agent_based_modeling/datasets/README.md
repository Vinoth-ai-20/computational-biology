# Module 15 — Datasets

All datasets in this module are either **synthetically generated** (in-notebook)
or **freely available public data**. No large files are committed to the repo —
raw data lives in `datasets/raw/` (gitignored) and is fetched by the scripts below.

| ID | Dataset | Source | Used in | Notes |
|----|---------|--------|---------|-------|
| DS01 | Synthetic SIR incidence (in-notebook) | Generated | NB01, NB09, NB12 | β=0.3, γ=0.1, N=10000; Poisson obs noise |
| DS02 | Synthetic Lotka-Volterra time series (in-notebook) | Generated | NB01, NB09 | r=1.0, a=0.1, b=0.1, d=1.5 |
| DS03 | COVID-19 daily incidence (UK, 2020) | [Our World in Data](https://ourworldindata.org/coronavirus) | NB09 (optional) | Use only for optional SIR fitting exercise |
| DS04 | NetworkX synthetic graphs (in-notebook) | Generated | NB07, NB12 | Erdős-Rényi G(N,p) and Barabási-Albert m=3 |
| DS05 | Synthetic gene expression noise data (in-notebook) | Generated | NB02 | Mimics Elowitz 2002 dual-reporter experiment |

## Fetch scripts

```bash
# DS03 — COVID-19 UK data (optional, ~50KB)
python scripts/fetch_covid_uk.py --output datasets/raw/covid_uk_2020.csv
```
