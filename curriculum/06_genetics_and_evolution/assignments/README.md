# Module 06 — Assignments

## A01 — Comparative Population Genetics

**Paired with:** NB04, NB06

**Brief:** Choose two human populations from 1000 Genomes summary statistics (e.g.
CEU vs. YRI). For 100 random SNPs:

1. Compute Fst for each SNP using the Weir-Cockerham estimator
2. Plot the Fst distribution
3. Identify the top 5 most differentiated SNPs — look up their genomic context
4. Test whether the top-10% Fst SNPs show signatures of selection (compare dN/dS
   proxy if functional annotation is available)

**Pass standard:** Working code that computes Fst correctly; a 2-panel figure
(Fst distribution + top SNP annotations); 1-paragraph biological interpretation.

## A02 — Phylogenetic Tree Comparison

**Paired with:** NB06, NB07

**Brief:** Using the primate cytochrome b alignment (DS03):

1. Build a UPGMA tree from Jukes-Cantor distances (implement from scratch)
2. Build a Neighbour-Joining tree from the same distances (implement from scratch)
3. Compare the two trees — do they agree? Where do they differ?
4. State one reason the two methods might disagree on this dataset

**Pass standard:** Both trees implemented correctly; visual comparison figure;
1-paragraph explanation of the discrepancy.
