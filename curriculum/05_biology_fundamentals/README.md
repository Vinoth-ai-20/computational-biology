# Module 05 — Biology Fundamentals

**Tier:** 1 — Master  
**Week budget:** Weeks 1–10 (parallel with Modules 01, 02, 03, 04)  
**Notebooks:** 18  
**Portfolio artifacts:**
- `portfolio/module05_hardy_weinberg_drift_simulator.png`
- `portfolio/module05_biology_for_computer_scientists.ipynb` (publishable explainer)

## Why this module is Tier 1

This is the real gap. Starting from zero formal biology on a 12-month clock means
this module runs in parallel from Week 1 — it cannot be deferred. Every downstream
module (genetics, genomics, transcriptomics, systems biology, ML for biology) requires
the vocabulary, mental models, and intuition built here. A Track A interviewer assumes
biology literacy; this module is what makes that assumption true.

## Objectives

By the end of this module you can:
- Explain the central dogma (DNA → RNA → protein) at the mechanistic level
- Compute Hardy-Weinberg genotype frequencies and test for deviations
- Simulate genetic drift with the Wright-Fisher model from scratch
- Draw and interpret a phylogenetic tree
- Explain cell signaling cascades (MAPK, Wnt) at the systems level
- Name and define the 50 vocabulary terms that appear in every biology paper
- Explain the biology in any of the 18 notebooks without notes (the Feynman bar)

## Why this helps Track A

Baseline biology literacy is assumed — not tested — in every RA/JRF/bioinformatics
interview. Interviewers use biology terminology without definition. This module is
the foundation that makes every other Track A module interpretable.

## Why this helps Track B

Every EMBL/IMPRS application requires a biological framing of your computational
project. You cannot write "I applied PCA to expression data" in a Track B statement
of purpose without also explaining what expression data is and what biological question
it addresses. This module provides that language.
