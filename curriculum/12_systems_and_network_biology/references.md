# Module 12 — References

## Core Textbooks

- **Alon — An Introduction to Systems Biology (2nd ed., 2019)**  
  The best single textbook for this module. Chapters on network motifs, robustness,
  and design principles are required reading at Tier 2.

- **Newman — Networks (2nd ed., 2018)**  
  Comprehensive graph theory reference. Chapters 6–10 cover degree distributions,
  centrality, and community detection at the right depth for Tier 2.

- **Barabási — Network Science (free online: networksciencebook.com)**  
  Highly visual; covers scale-free networks and biological applications.

- **Palsson — Systems Biology: Constraint-based Reconstruction and Analysis (2015)**  
  Reference for FBA and metabolic network modelling.

## Key Software

- **NetworkX:** pip install networkx — standard Python graph library
- **COBRApy:** pip install cobra — FBA and metabolic network analysis
- **igraph (Python):** pip install igraph — faster for large networks
- **scipy.integrate.solve_ivp:** ODE solving (in scipy, no extra install)

## Key Databases

- **STRING (string-db.org):** PPI networks with confidence scores; download 9606 (human)
- **BioGRID (thebiogrid.org):** curated experimental PPI interactions
- **DisGeNET (disgenet.org):** disease-gene associations
- **KEGG Pathway (kegg.jp):** metabolic and signaling pathways
- **BiGG Models (bigg.ucsd.edu):** genome-scale metabolic models (download iJO1366 for E. coli)
- **OMIM (omim.org):** Online Mendelian Inheritance in Man — disease-gene links
- **Reactome (reactome.org):** curated pathway database
