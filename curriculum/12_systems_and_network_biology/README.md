# Module 12 — Systems and Network Biology

**Depth tier:** 2 — Working competence (Promotion candidate — revisit at Month-3 checkpoint)  
**Week budget:** Weeks 31–37 (7 weeks)  
**Notebooks:** 12  
**Mini-project:** Yes (MP01 — PPI network analysis pipeline)

## Objectives

Reach working competence in network-theoretic analysis of biological systems:
build and analyze biological networks from scratch, implement key graph algorithms,
understand ODE-based dynamical models, and perform a basic flux balance analysis.
Can use the standard tools (NetworkX, COBRApy) correctly and explain what they do.

## Why this helps Track A

The Tier 2 promotion candidate note reflects real 2026 India job postings for
"Computational Systems Biologist" and "Network Biology Analyst" roles at CSIR, IISc,
and biotech companies. Specifically tested: network centrality, PPI network analysis,
pathway enrichment, graph-based clustering. NetworkX and COBRApy fluency are explicitly
listed as desirable skills. This module covers all of them.

## Why this helps Track B

Systems & network biology is central to EMBL-EBI, Wellcome Sanger, and multiple
IMPRS programmes. The Turing Institute's "AI for biological networks" theme maps
directly to NB05–NB07. Boolean network models and ODE systems (NB08–NB09) are
prerequisite for computational neuroscience and ALife tracks.
