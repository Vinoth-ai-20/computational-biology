# Module 13 — References

## Textbooks

- **The Elements of Statistical Learning** — Hastie, Tibshirani, Friedman (2009); free PDF at web.stanford.edu/~hastie/ElemStatLearn/; the standard reference for statistical ML theory
- **Pattern Recognition and Machine Learning** — Bishop (2006); Bayesian perspective; dense but thorough
- **An Introduction to Statistical Learning** (ISL) — James, Witten, Hastie, Tibshirani (2021); accessible companion to ESL with R/Python labs; free PDF at statlearning.com
- **Deep Learning** — Goodfellow, Bengio, Courville (2016); chapters 1–6 cover foundations shared with Module 13

## Biology-Specific ML Books

- **Biological Sequence Analysis** — Durbin, Eddy, Krogh, Mitchison (1998); probabilistic models for sequence data; foundational for genomics ML
- **Computational Genome Analysis** — Deonier, Tavaré, Waterman (2005); statistical approaches for genomic data
- **Deep Learning for the Life Sciences** — Ramsundar et al. (2019, O'Reilly); practical deep learning for drug discovery, genomics, microscopy

## Key Software

- **scikit-learn** — sklearn.org; the standard Python ML library; exhaustive documentation
- **XGBoost** — xgboost.readthedocs.io; gradient boosted trees; dominant for tabular biological data
- **LightGBM** — lightgbm.readthedocs.io; faster GBDT alternative
- **imbalanced-learn** — imbalanced-learn.org; SMOTE and other class imbalance methods
- **SHAP** — shap.readthedocs.io; model-agnostic feature importance / explainability

## Key Biological ML Tools

- **DeepBind** — sequence-specific binding prediction (Alipanahi et al. 2015)
- **CADD** — Combined Annotation-Dependent Depletion; variant pathogenicity scores
- **DANN** — deep learning variant effect predictor
- **PredPrins** — protein function prediction
